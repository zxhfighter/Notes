- python 版本可以多版本共存
- 一些机器会预装 python3（或 python2），检测是否安装的命令如下：

```sh
# or python for python 2
$ python3
```

- 退出 REPL 交互，输入命令 `exit()` 或者 `quit()` 即可

- mac 安装从 http://python.org/download/ 这里下载安装包，点击安装即可

- 使用 `help(object)` 查看对象的 API 文档

```sh
$ help(print)
$ help(type)
$ help(isinstance)
$ help(str.split)
$ help(str.format)
```

- 编辑器可以使用 vscode 即可，**调试也可以使用**
